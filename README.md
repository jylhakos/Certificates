## Courses

### **[Big Data Platforms](certificate-big-data-platforms.pdf)**
Certificate covering big data technologies, distributed computing systems, PySpark and data processing frameworks.

### **[Containers](certificate-containers.png)**
Training on container technologies, containerization principles, Docker, and container orchestration basics.

### **[Continuous Integration](certificate-continuous-integration.pdf)**
Course focused on CI/CD pipelines, automated testing, build automation, Jenkins, Git Ops, and continuous delivery practices.

### **[CUDA and GPU Programming](certificate-cuda-and-gpu.pdf)**
Advanced course on neural networks, deep learning frameworks, machine learning algorithms, GPUs, CUDA, Python and C++.

### **[Databases](certificate-databases.pdf)**
Training covering database design, SQL, NoSQL databases, PostgreSQL, and database management systems.

### **[Deep Learning](certificate-deep-learning.pdf)**
Course on neural networks, deep learning algorithms, Python, Keras and Tensorflow.

### **[Docker](certificate-docker.pdf)**
Specialized certification in Docker containerization, Docker Compose, and container management.

### **[Full Stack Development](certificate-fullstack.png)**
The full-stack web development course covering frontend with React and backend technologies with Node.js.

### **[GraphQL](certificate-graphql.png)**
Course on GraphQL API design, queries, mutations, and GraphQL server implementation.

### **[Kubernetes](certificate-kubernetes.pdf)**
Training on Kubernetes container orchestration, cluster management, Google GCP and cloud-native deployments.

### **[React](certificate-react.pdf)**
Certificate in React.js library, component architecture, state management, and modern React patterns.

### **[React Native](certificate-reactnative.png)**
Mobile application development using React Native framework for iOS and Android platforms.

### **[Software Technologies](Software%20Technologies.pdf)**
Overview of software technologies, development tools, programming paradigms, and industry best practices for building scalable applications.

### **[TypeScript](certificate-typescript.pdf)**
Course covering TypeScript language, type systems, HTML, CSS and TypeScript in modern web development.
